Policy Interest Rate Simulator — First Run Instructions

macOS:
	1	Go to PIRS_mac and double click PIRS_mac.app.

	2	If you see a message like “PIRS_mac.app is from an unidentified 		developer”, don’t worry. This happens because the game is not signed 		with Apple’s paid developer license.

	3	Right click the app, choose “Open”, then click “Open” again.

	4	This is only needed once. After that you can open it normally.


Windows 10:
	1	Go to PIRS_windows and double click PIRS_windows.exe.

	2	If you see a blue screen “Windows protected your PC”, don’t worry. 		This happens because the game is not signed with Microsoft’s paid 		certificate.

	3	Click “More info”.

	4	Click “Run anyway”.

	5	This is only needed once. After that you can open it normally.


Windows 11:
	1	Go to PIRS_windows and double click PIRS_windows.exe.

	2	If you see a blue screen “Windows protected your PC”, don’t worry. 		This happens because the game is not signed with Microsoft’s paid 		certificate.

	3	Right click PIRS_windows.exe->properties
	
	4	At the bottom, you will find a security disclosure warning you that 		the file came from another computer. There is a unblock box there, 		click on it.

	4	Close and double click PIRS_windows.exe, the game should start

	5	This is only needed once. After that you can open it normally.



Charts you save in the game will appear in your user folder:
	•	macOS: ~/EconGame/charts/
	•	Windows: C:\Users\YOURNAME\EconGame\charts\


That’s it — enjoy the game!
